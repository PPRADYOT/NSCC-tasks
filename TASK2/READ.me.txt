# 🔹 Task 2 – Interactive Quiz App  

**Objective:** Create a 5-question multiple-choice quiz.  

---

## 📌 Features  

- 5 questions, each with 4 options  
- Only one option can be selected per question  
- Submit button calculates results  
- Displays final score → *“You scored X out of 5”*  
- **Brownie Subtask:** Shows which answers were correct/incorrect after submission  

---

## ⚡ How to Run  

```bash
# open project folder
cd Task-2-Quiz-App

# open in browser
open index.html   # or double-click index.html
